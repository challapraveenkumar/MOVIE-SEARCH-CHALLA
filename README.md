🎬 Movie Search Web Application

This is a React.js movie search web application that allows users to search for movies and view detailed information such as ratings, release year, and plot. The application fetches real-time movie data from the OMDb API and displays it in a clean and responsive user interface.

🚀 Features

🔍 Search for movies by title

⭐ View movie ratings and details

📅 Display release year and movie poster

⏳ Loading state while fetching data

⚠️ Error handling for invalid searches

📱 Responsive design for different screen sizes

🛠️ Technologies Used

React.js (Frontend Library)
JavaScript (ES6+)

CSS3 for styling

OMDb API for movie data

📌 How It Works

User enters a movie name in the search bar.

The app sends a request to the OMDb API.

Movie data is fetched and displayed instantly.

Users can view ratings, poster, and additional movie details.
