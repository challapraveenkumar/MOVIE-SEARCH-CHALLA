🎬 Movie Fetching App (React JS)

This is a Movie Search Application built using React.js that fetches movie data from the OMDb API. Users can search for movies and view details like rating, year, and poster in a clean and responsive UI.

🚀 Features

🔍 Search movies by title

🎥 Fetch movie data from OMDb API

⭐ Display movie ratings and details

🖼 Show movie posters

⚡ Fast and responsive React UI

🔄 Real-time API data fetching

🛠 Tech Stack

React.js

JavaScript (ES6)

CSS

OMDb API
